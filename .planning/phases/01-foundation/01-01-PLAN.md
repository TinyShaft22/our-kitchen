<task type="auto">
  <name>Task 2: Configure Tailwind CSS with custom theme</name>
  <files>tailwind.config.js, postcss.config.js, src/index.css</files>
  <action>
Install Tailwind: `npm install -D tailwindcss postcss autoprefixer && npx tailwindcss init -p`

Configure tailwind.config.js:
- content: ["./index.html", "./src/**/*.{js,ts,jsx,tsx}"]
- Extend theme.colors with custom palette:
  - cream: '#FDF8F3' (background)
  - terracotta: '#C4755B' (primary)
  - sage: '#87A878' (secondary)
  - honey: '#E8B86D' (accent)
  - charcoal: '#3D3D3D' (text)
- Extend theme.borderRadius with: soft: '12px', softer: '16px'
- Extend theme.boxShadow with: soft: '0 2px 8px rgba(0,0,0,0.08)'
- Extend theme.fontFamily with: sans: ['Inter', 'system-ui', 'sans-serif']

Update src/index.css:
- Add Tailwind directives (@tailwind base/components/utilities)
- Add base styles: body { @apply bg-cream text-charcoal; }

Update App.tsx to use Tailwind classes: bg-cream, text-charcoal, rounded-soft, etc.
  </action>
  <verify>npm run dev shows warm cream background with charcoal text, Tailwind classes work</verify>
  <done>Tailwind configured with all custom theme values, base styles applied</done>
</task>
